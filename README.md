# Cats App

## Install Dependencies 💾

```bash
# NPM
npm install

# or YARN
yarn install

# or PNPM
pnpm install

# or BUN
bun install
```

## Run Development Mode 🧑‍💻

```bash
# NPM
npm run dev

# or YARN
yarn dev

# or PNPM
pnpm dev

# or BUN
bun dev
```

## Build Project 👷🏗️

```bash
# NPM
npm run build

# or YARN
yarn run build

# or PNPM
pnpm run build

# or BUN
bun run build
```

## Start Production Mode 🌎

**Remember, in order to use this script you must build the project before 😉**

```bash
# NPM
npm start

# or YARN
yarn start

# or PNPM
pnpm start

# or BUN
bun start
```
